# UniCase

To rozszerzenie zapewnia płynną konwersję tekstu na różne wielkości liter, w tym wielkie i małe litery, wielkość liter w tytule i wielkość liter. Pożegnaj ręczne dostosowywanie wielkości liter i podnieś jakość kodowania dzięki rozszerzeniu UniCase dla Visual Studio Code

##

[![Rozszerzenie Vscode](/translations/demo.gif 'Demo rozszerzenia Vscode')](https://learnwithyan.com)

## Jak używać?

Otwórz dokument tekstowy. Wybierz tekst do konwersji. Zacznij pisać „Unicase”. Tutaj możesz zobaczyć menu poleceń

## Jakich poleceń możesz użyć?

- UniCase - przetłumacz plik Readme
- UniCase - tekst pisany WIELKIMI LITERAMI
- UniCase - tekst na małe litery
- UniCase - tekst do tytułu
- UniCase - tekst na wielkość zdania
- UniCase - tekst na zMiEnNe pRzYpAdKi
